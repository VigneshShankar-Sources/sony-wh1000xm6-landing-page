# Sony WH-1000XM6 Landing Page

A hyper-premium, Apple-level cinematic scrollytelling landing page for Sony WH-1000XM6 headphones. Built with Next.js 14, Tailwind CSS, and Framer Motion. This project features a 240-frame image sequence powered by an HTML5 Canvas that is fully synchronized with scroll position to animate the assembly and disassembly of the headphones.

## How to Run the Project

1. Clone the repository
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Rendering**: HTML5 Canvas
